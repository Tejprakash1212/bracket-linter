# Bracket Linter — Real-Time Syntax Checker

A real-time bracket validation web app built using Python and Streamlit based on Context-Free Grammar (CFG) and Pushdown Automata (PDA) concepts.

## 🚀 Features
- Detects mismatched and unclosed brackets
- Real-time syntax checking
- Stack-based parsing
- O(n) time complexity

## 🛠️ Tech Stack
- Python
- Streamlit
- Stack Data Structure

## 📚 Concepts Used
- Context-Free Grammar (CFG)
- Pushdown Automata (PDA)
- Automata Theory
- Lexical Scanning

## ▶️ Run Project
```bash
streamlit run app.py
```
